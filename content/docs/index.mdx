# Welcome to Full Stack Docs ⚡

Welcome to the comprehensive developer documentation. This hub is built as a single source of truth for full-stack engineering practices, software development principles, and system design patterns.

Whether you are building interactive web frontends, scaling robust JVM microservices, fine-tuning SQL queries, or designing distributed networks, this documentation serves as your reference guide.

---

## Roadmap Chapters

Here is an overview of the domains covered in this roadmap:

### 🌐 Frontend Engineering
Learn semantic markup, layout systems, responsive UI design, performance optimizations, and state lifecycle management using React.

*   [**HTML**](/docs/html/introduction-to-html) — Elements, Forms, Accessibility (a11y), and HTML5 APIs.
*   [**CSS**](/docs/css/introduction-to-css) — Specificity, Box Model, Flexbox/Grid, Responsive Architecture, and Variables.
*   [**JavaScript**](/docs/javascript/introduction-setup) — Closures, Prototypes, Event Loop, Async/Await, and Modules.
*   [**React**](/docs/react/introduction-setup) — Components, Hooks, Context, State Management, and Performance.

### 🔧 Version Control & Collaboration
Learn local version tracking, staging mechanics, branch management, remote syncing, and collaborative workflows.

*   [**Git**](/docs/git/introduction-to-git) — History, branching, stash, rebase, and conflict resolution.
*   [**GitHub**](/docs/github/introduction-to-github) — SSH keys, PR workflows, reviews, and Actions CI/CD.

### ☕ Backend Development & Databases
Understand strongly typed object-oriented programming, MVC architectures, dependency injection, caching, and data modeling.

*   [**Java**](/docs/java/introduction-setup) — JDK fundamentals, OOP, Collections, Streams, and JVM memory.
*   [**Spring Boot**](/docs/spring-boot/introduction-setup) — Dependency Injection, Spring Core, REST API, Security, and JPA.
*   [**Hibernate / JPA**](/docs/hibernate-orm-jpa/introduction-to-orm) — ORM patterns, relations, caching, and optimization.
*   [**MySQL**](/docs/mysql/introduction-setup) — Queries, indexing, ACID transactions, and query optimization.
*   [**PostgreSQL**](/docs/postgresql/introduction-setup) — JSONB, advanced indexing, window functions, and scaling.

### 🚢 Infrastructure & Architecture
Scale your applications to handle millions of requests, run automated builds, and configure distributed systems.

*   [**Docker**](/docs/docker/introduction-to-containers) — Containers, Images, Dockerfile, Networks, and Volumes.
*   [**System Design**](/docs/system-design/fundamentals) — Load Balancing, Caching, Sharding, Message Queues, and Case Studies.
